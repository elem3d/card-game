# Po-Kém-Pô
#### This JavaScript project made in 2023 is a classic game of jo-kem-pô (aka rock-papper-sisors) with the pokémon theme, perfect for fans!
- Custom cursor;
- Imersive soundtrack;
- Play against computer;
- Five different Pokemons to choose
